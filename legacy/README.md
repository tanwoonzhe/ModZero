# legacy/

This directory contains code preserved from earlier iterations of the project.
It is kept for reference and is not part of the active ZTNA architecture.

| Directory          | What it was                                      |
|--------------------|--------------------------------------------------|
| `client/`          | Tauri-based desktop client (replaced by Electron `client-app/`) |
| `intranet/`        | Demo Nginx mock resource (still used by dev compose) |
| `scripts/`         | Utility and regression test scripts              |
| `zerotrustassemetment/` | Microsoft NIST 800-207 reference assessment |
| `root-fixtures/`   | Test JSON/HTML fixtures from the project root    |
